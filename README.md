# 🚀 Aurevia's AtherList

Aurevia's AtherList is a gamified productivity web application designed to help users manage tasks efficiently while staying motivated through XP rewards, level progression, achievements, streaks, and productivity tools.

## 📌 Project Overview

This project was developed as part of an internship task to create an engaging and user-friendly task management system. It combines traditional to-do list functionality with gamification elements to encourage consistency and productivity.

## ✨ Features

### 🔐 Authentication

* User Signup
* User Login
* Password Validation
* Secure User Access

### 📋 Task Management

* Create Tasks
* Edit Tasks
* Delete Tasks
* Mark Tasks as Completed
* Task Categories
* Due Dates
* Task Search
* Task Filtering
* Progress Tracking

### 🎮 Gamification System

* XP Rewards for Completing Tasks
* Level Progression System
* Achievement Badges
* Trophy Collection
* Daily Streak Tracking
* Productivity Motivation System

### ⏱️ Productivity Tools

* Built-in Pomodoro Timer
* Focus Sessions
* Time Management Support

### 🎨 User Experience

* Dark Mode
* Light Mode
* Clean and Modern Interface
* Fully Responsive Design
* Mobile, Tablet, and Desktop Compatible

### 💾 Data Storage

* Browser Local Storage
* Persistent User Data
* No External Database Required

## 🛠️ Tech Stack
