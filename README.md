# 🚀 Aurevia's atherList - Gamified To-Do Web App

Aurevia's atherList is a gamified productivity web application that helps users stay organized and motivated by combining task management with rewards, XP, levels, streaks, and achieveme

✨ Features

### 📋 Task Management
- Add, edit, complete, and delete tasks
- Task prioritization
- Due dates for tasks
- Search and filter tasks
- Progress tracking

### 🎮 Gamification
- Earn XP for completing tasks
- Level progression system
- Achievement badges and trophies
- Daily streak tracking
- User progress dashboard

### ⏱️ Productivity Tools
- Built-in Pomodoro Timer
- Focus session support
- Productivity enhancement features

### 🎨 User Interface
- Light and Dark Mode
- Responsive design for mobile, tablet, and desktop
- Smooth and modern user experience

### 💾 Data Storage
- Uses Browser Local Storage
- No backend required
- User data persists between sessions on the same device

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- React.js
- Tailwind CSS
- Vite
